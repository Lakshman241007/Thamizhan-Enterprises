🚆 Railway Data Management System

This project is a web-based railway data management platform developed to monitor train routes, delays, and operational logs efficiently. The system helps teams manage railway operations through a structured and user-friendly dashboard.

Currently the system includes modules such as data entry, route tracking, progress monitoring, and operational logs. More intelligent features and analytics tools will be added as the project evolves.
